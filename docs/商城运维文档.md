# 商城运维文档

## 如何配置菜单

### 微信配置

* token令牌   (wx.token) 
* 启用消息加密  (wx.is_encrypt_message)
* 消息加解密密钥  (wx.encoding_aes_key)
* 应用ID (wx.app_id)
* 应用密钥 (wx.app_secret)
* 域名 (wx.host)
* 商户ID (wx.partner_id)
* 商户Key (wx.partner_key)
* 小程序AppId (wx.wxa_appid)
* 小程序密钥 (wx.wxa_app_secret)
* 登录自动注册 (wx.auto_reg)
